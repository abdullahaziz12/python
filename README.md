A collection of Python programs written while learning the fundamentals of Python programming. This repository contains small, independent scripts covering core programming concepts such as variables, data types, loops, functions, object-oriented programming, file handling, exception handling, and more.

## 📚 About

This is not a single project—it's a growing collection of standalone Python programs created while practicing different programming concepts. Each file focuses on a specific topic or problem to strengthen programming fundamentals and problem-solving skills.

## 🛠️ How to Run

1. Clone this repository.
2. Open the project in Visual Studio Code or any Python IDE.
3. Run any Python file using:

```bash
python filename.py
```

Replace `filename.py` with the program you want to execute.
